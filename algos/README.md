# Algorithms

A collection of common algorithms implemented in JavaScript and Python.

## Implemented Algorithms

- **[Fibonacci](./fibonacci/)**: Generates the Fibonacci sequence.
- **[Insertion Sort](./insertion-sort/)**: A simple, comparison-based sorting algorithm.
- **[Maximum Subarray](./maximum-subarray/)**: Finds the contiguous subarray with the largest sum.
- **[Merge Sort](./merge-sort/)**: An efficient, divide-and-conquer sorting algorithm.
- **[Strassen's Matrix Multiplication](./strassens-matrix-multiplication/)**: A divide-and-conquer algorithm for fast matrix multiplication.
